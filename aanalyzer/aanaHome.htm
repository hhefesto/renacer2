AAnalyzer®: a peak-fitting program for photoemission data

  • AAnalyzer®: the non-complicated software for peak-fitting photoemission data:
    • It can be used for easy and difficult data (the latter illustrated in the figures below).
    • It is very easy to use as new users become experts in little time.
  • State of the art methods:
    •  Allows for an excellent modeling of the background.
    •  Allows for a precise assessment of the area of the peaks.
  • Right physics:
    •  All the fitting parameters have direct physical interpretation.
    •  Employs true Voigt functions, among others.
