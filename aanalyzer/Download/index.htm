AAnalyzer1.07 is an application for Photoemission and Infrared Analysis. You are allowed to download a completely free and functional version of AAnalyzer that expires 150 days after the installation.

Installation instructions

  1. Execute the file AAnalyzerInstaller.exe and choose a folder on your computer to save the installation files
  2. Execute the saved file setup.exe and follow the instructions of the Installation Wizard.
  3. To uninstall AAnalyzer go to the Control Panel and open Add/Remove Software then select AAnalyzer.

Minimum system requirements

  1. CPU: Pentium I at 100MHz (equivalent).
  2. RAM: 516MB.
  3. DISK: 4MB of free space.
  4. DISPLAY: VGA with a 800x600 resolution and 256 colors.
  5. Platform: Windows 9x, ME, NT, 2000, XP and Vista.
  6. To install this software in Win NT, 2000 and XP platforms you will need administrative privileges.
